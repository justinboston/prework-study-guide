HTML

  • The head element contains information, or metadata, that helps the browser to render the page correctly.
  • The body element contains the content that is shown to the user.
  • The !DOCTYPE html declaration. This declaration tells the browser what document type it should expect when it is opened.
  • The title element defines the title for a webpage. This element is required and is extremely important for SEO.
  • The h1, h2, h3, h4 elements represent the level of heading a given text block.
  • The p element represents a paragraph or block of text.
  • The ul, ol, li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage. The src attribute is the most important because it defines the location of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • To hyperlink the a element (which stands for anchor) creates links to a webpage. The href attribute points to the URL for the link.

CSS

  • There are three ways to style a webpage using CSS: Inline CSS, Internal CSS Style Sheet & External CSS Style Sheet
  • Inline CSS = Styling directly within the HTML elements, not best practice
  • Internal CSS style sheet, applies CSS rules for the webpage in the head section of the HTML file
  • External CSS style sheet = Linking to an external CSCS file, typically used as a best practice
  • Link an external style sheet using link rel = "stylesheet" href="stylesheetname.css"
  • * is a Universal Selector which is used to apply a rule to all the elements visible on the page.
  • A period preceding a html class defines it as a Class Selector
  • Commas can be used within typical selectors can be to apply a CSS rule to multiple classes of elements w/in the HTML

JavaScript

  • Using JavaScript, you can power your websites with effects, interactivity, data communication, and much more.
  • Whereas the style.css file is linked within the head element, the script.js file is linked within the body element.
  • Use script src="script.js" to link a JavaSript file w/in an HTML document.
  • A variable is a named container that allows us to store data in our code.
  • The console.log method was created to make it easier for developers to test their code.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
